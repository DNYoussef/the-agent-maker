================================================================================
                          INTERACTION EXAMPLES
================================================================================

1. ROTATE VIEW:
   - Click and drag anywhere on the 3D plot
   - Camera rotates around center of tree
   - Can view tree from any angle (top, side, isometric)

2. ZOOM:
   - Scroll wheel up = zoom in
   - Scroll wheel down = zoom out
   - Useful for examining specific generations

3. PAN:
   - Hold Shift + drag
   - Moves camera position
   - Useful for centering specific regions

4. HOVER:
   - Move mouse over any node
   - Tooltip appears with model details
   - Shows: ID, generation, fitness, diversity, technique, parents, size

5. SELECT LINEAGE:
   - Choose model from dropdown above visualization
   - All ancestor nodes highlighted in amber
   - White borders added to lineage nodes
   - Other nodes dimmed for clarity

6. FILTER GENERATIONS:
   - Adjust slider below dropdown
   - Only selected generation range shown
   - Statistics update to reflect filtered data

7. REGENERATE:
   - Click "Regenerate Tree" button
   - New random evolutionary tree generated
   - Different techniques, fitness trajectories
   - Useful for demonstrating variability
